PROJECT GOODMAN

EDUCATION FOR ALL

ABOUT US

Project GoodMan is a browser extension that mines Monero Cryptocurrency from User's CPU Power. Mined crypto from user's browser is donated to build schools and distribute books to/for weaker sections of society. Project GoodMan aims to present every member of society with better reach to education, thus helping to eradicate poverty to some extent.

HOW IT WORKS?

Project GoodMan works in 3 simple ways:

Installation

Install our extension on your browser in easy steps (All browsers supported)

Mining

GoodMan extension starts mining Monero with every new Session (You have an option to stop mining)

Donations

Mined Crypto is credited to Project GoodMan's Crypto Wallet Donations are made to build schools, distribute books and more...
